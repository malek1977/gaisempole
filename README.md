# وكيل الذهب الذكي - بوت تليجرام 🪙

## المميزات ✨

- 🪙 أسعار الذهب بـ 3 عيارات (18, 21, 24 قيراط)
- 💱 3 عملات (USD, SAR, YER)
- 📊 أسعار الجرام وأجزائه
- 📰 تحليل المشاعر الإخبارية
- 📈 تقرير COT
- 🏙️ فرص المراجحة المحلية في اليمن
- ⚠️ إدارة المخاطر

## الأوامر 📋

| الأمر | الوظيفة |
|-------|---------|
| `/start` | الترحيب والأوامر |
| `/help` | المساعدة |
| `/report` | التقرير الشامل |
| `/gold` | سعر الذهب العالمي |
| `/prices` | جدول العيارات |
| `/arbitrage` | فرص المراجحة |
| `/risk` | إدارة المخاطر |
| `/status` | حالة البوت |

## النشر على Render 🚀

1. أضف المتغيرات البيئية
2. اربط مستودع GitHub
3. اختر Python 3
4. أمر البناء: `pip install -r requirements.txt`
5. أمر التشغيل: `python main.py`
